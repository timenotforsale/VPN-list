## EEVPN


- 注册入口：[点击注册（EEVPN）](https://www.eevpn66.com/#/register?code=Lshm2ta4)


### 简评

- 这家确实更偏轻度使用，Basic 是最便宜的入门档，流量少，适合偶尔用的人。
- Standard 才是更适合长期当主力的档位，100GB/月的空间更均衡。
- Premium、Max、Ultra 主要是给流量明显更大的人准备的，价格随着流量上去得也比较快。

## SKTLUMO

- 注册入口：[点击注册（SKTLUMO）](http://163.192.42.74/#/register?code=DtkuCul1)

### 简评

- 这个反而是适合买流量套餐而不是周期套餐的了。
- 经典老套餐一次性很大，适合超大流量需求，但价格也高（不过最近有半价优惠），属于一次性重度方案。
- 旗舰套餐是最明显的强档，999G/月让它在“大流量 + 价格”组合上很突出。
- 基础套餐和高级套餐更像日常档位，基础套餐便宜，高级套餐在设备数和带宽上更均衡。
- 如果只挑一个，旗舰套餐最像“综合性价比”方案；如果按低预算，基础套餐更友好。
